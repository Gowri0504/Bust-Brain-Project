# Backend

## Env
- Use `.env` based on `.env.example`.

## Run
- `npm i`
- `npm start`

## APIs
- `GET /auth/airtable/login`
- `GET /auth/airtable/callback`
- `GET /bases`
- `GET /bases/:bid/tables`
- `POST /forms`
- `GET /forms/:id`
- `POST /forms/:id/submit`
- `GET /forms/:id/responses`
- `POST /webhooks/airtable`
